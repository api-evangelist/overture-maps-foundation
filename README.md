# Overture Maps Foundation (overture-maps-foundation)
The Overture Maps Foundation is a Linux Foundation project created in partnership with Meta, Microsoft, AWS, and TomTom to build reliable, open map data as an alternative to proprietary mapping platforms. It provides openly available datasets for places, buildings, transportation, and administrative boundaries.
